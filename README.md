# BTS - Bluetooth GPS Tracker
* React-Native
* Expo
* Kotlin
* Yarn

## Platform Tested
- [x] Android
- [ ] IOS
- [ ] Web
- [ ] Desktop

## Features
- [x] Show Bluetooth Devices
- [x] Track Connected Bluetooth Devices
- [x] Save last Connected Bluetooth Devices
- [x] Open Geo Saved on Map
- [ ] Have a Personal Map
- [ ] Track GPS when Connect on Background Process

## Exec
### Install
```
  yarn install
```
### Run
```
  yarn start
```
### Run Android (USB ADB)
```
  yarn android
```
### Pre-Build
```
  yarn prebuild
```
